# pawsome_pup_directory
Pawsome Pup Directory Code Challenge

Uses https://dog.ceo/dog-api/

To start:

1. Clone Repository
2. Run
    ```
    npm start
    ```

Dependencies
 - node.js
 - Bootstrap
 - Angular
 - jQuery
 - popper.js
