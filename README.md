# pawsome_pup_directory
Pawsome Pup Directory Code Challenge

Implements https://dog.ceo/dog-api/

To start:

1. Clone Repository
2. Run
    ```
    npm start
    ```
3. Open at http://localhost:4200/

Dependencies
 - node.js
 - Bootstrap
 - Angular
 - jQuery
 - popper.js
